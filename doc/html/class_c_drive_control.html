Detailed Description

Осуществляет управление движением робота

Реализует как простые функции движения (ехать прямо, повороты), так и алгоритм перемещения до следующей синей полосы со смещением на определенной количество черных полос всторону (алгоритм движения к фишке).

Движение робота к фишке

Необходимо вызывать в цикле до конца работы алгоритма. Движение контролируется на основе данных от датчиков линии.

Parameters
[in]leftEnПоказание левого датчика линии: "0" - белое поле; "1" - черная линия
[in]rightEnПоказание правого датчика линии: "0" - белое поле; "1" - черная линия
Returns
код завершения конректного этапа алгоритма, необходимый для перехода, например, к алгоритму движения по черной линии
