\doxysubsection{Detailed Description}
Класс работы с датчиком цвета на T\+C\+S3200. 

Отвечает за определение поля (1 или 2(!)) -\/ распознавание цвета на старте, определение достижения синей полосы и зоны финиша (зеленого цвета) \begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000001}{Todo}}]Диапазон чувствительности у разных цветов датчика может быть разным, поэтому стоит их привести к общему знаменателю\+: green\+Color = map(green\+Frequency, 100, 199, 255, 0); \end{DoxyRefDesc}
